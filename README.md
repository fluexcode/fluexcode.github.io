# 🎯 Kick.com Bot Kontrol Paneli

Kick.com için Python botu ve web kontrol paneli.

## 📋 Özellikler
- 7/24 çalışan bot (GitHub Actions)
- Web tabanlı kontrol paneli
- Özel komutlar (!merhaba, !saat vb.)
- Otomatik mesajlar
- Canlı chat takibi

## 🚀 Kurulum

1. Bu repoyu forkla
2. GitHub Secrets'a ekle:
   - `KICK_USERNAME`: Email adresin
   - `KICK_PASSWORD`: Şifren
   - `KICK_CHANNEL`: Kanal adın
3. GitHub Pages'i aktifleştir
4. Bot otomatik çalışmaya başlar!
